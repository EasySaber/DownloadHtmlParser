**********************
Описание:
-Приложение скачивает HTML страницу, посредством https запроса.
-Подсчитывает количество уникальных слов на странице, выводит результат в консоль.

**********************
Требования для запуска приложения:
-Установленная JDK 8+ (Для работы приложения)
-SQLite (Для просмотра базы данных со словами. Опционально)

**********************
Запуск приложения(варианты на выбор):
-URLDownload-2.bat
-команда: java -jar URLDownload-2.0-jar-with-dependencies.jar в терминале

**********************
Проверка работы приложения:
-Введите адрес формата https:\\www.********
-Нажмите Enter
>>>>>>>>>>
Пример:
https://www.java.com/ru/
https://ru.wikipedia.org/wiki/Java
https://yandex.ru/search/?text=программирование&lr=195&clid=9582&src=suggest_B
>>>>>>>>>>
-Для завершения работы с приложением введите вместо адреса "+" и нажмите Enter

**********************
После выполнения запроса формируются:
-Download.html - сохраненная HTML страница, по Вашему запросу 
-DownloadNoTag.txt - файл, в котором содержиться весь видимый текст, без HTML-тегов
-Words.txt - полный список слов, найденных на странице
-SearchWords.db - сформированная БД с подсчетом количества уникальных слов на странице.
-Result.txt - сформированный текстовый файл с подсчетом количества уникальных слов на странице.

**********************
Обратите внимание: после каждого нового запроса предыдушие результаты сохраняются в файле Result.txt.
